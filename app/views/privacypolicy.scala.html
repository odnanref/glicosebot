Política de privacidade

Relatório de glicose

Os dados do utilizador e informação fornecida por ele serão guardados por nós para usufruto do mesmo,
ficando desde já o utilizador com esse conhecimento e que os dados não se encontram encriptados na nossa base de dados. GlicoseBot pode usufruir desses mesmos dados para análise e estatísticas de forma anónima
sem identificação do utilizador específico. Os dados que são fornecidos pelo facebook sobre o utilizador (primeiro nome, último nome, foto
, gênero, hora local, id do utilizador no glicosebot, estes dados são também guardados na nossa
base de dados, juntamente com os dados fornecidos pelo utilizador e registos de conversa.
O Facebook tem acesso a toda a informação que passa pelo facebook messenger, serviço do qual o nosso bot depende.

Na internet os sistemas que a compõe não são 100% seguros e apesar de termos métodos para garantir
a segurança dos dados e do caminho que os comunica, não é possível garantir a sua segurança a 100%.
